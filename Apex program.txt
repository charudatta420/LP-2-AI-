Factorial Apex code

Integer n = 5;
Integer fact = 1;
for(Integer i = 1; i <= n; i++){
    fact = fact * i;
}
System.debug('Factorial of ' + n + ' = ' + fact);




Fibonacci series Apex code

Integer n = 10;
Integer a = 0, b = 1, c;
System.debug('Fibonacci Series:');
System.debug(a);
System.debug(b);
for(Integer i = 2; i < n; i++){
    c = a + b;
    System.debug(c);
    a = b;
    b = c;
}
